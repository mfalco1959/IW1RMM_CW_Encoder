Capabilities

What it does

🔑

WinKey WK3 Protocol

Full WK3 binary protocol over USB Serial at 1200 baud. Compatible with N1MM+, Win-Test, DXLog, Logger32, WKdemo, WK3demo.

📡

BLE K3NG Control

Nordic UART BLE service for wireless control from Android or PC. K3NG ASCII command set with backslash commands. Always available regardless of WinKey mode.

🖐

All Keying Modes

Iambic A, Iambic B, Manual (straight key), Auto (electronic bug), Ultimatic. Real-time character decoding on display.

📺

LVGL Touchscreen UI

5-tab graphical interface on ILI9341 320×240 display. Interactive 4-point touch calibration on first boot. NEW v7.1.4

💾

SD Card Playback

Load .txt files from MicroSD and transmit in Morse at configurable speed. PLAY / PAUSE / STOP controls with real-time text display.

⚡

HSCW & QRSS

High Speed CW (60–100+ WPM) for meteor scatter and aurora. QRSS modes (3s, 6s, 10s, 30s per DIT) for low-power propagation experiments.

📻

6 Preset Messages

CQ, Name, Test, Ant/Rig, RST, Free — editable via on-screen keyboard, stored permanently in NVS flash.

🔁

FreeRTOS Multitasking

morseTask on Core 0 for precise timing. LVGL UI on Core 1. 1 kHz hardware timer for sub-millisecond accuracy. Non-blocking architecture throughout.

🏆

Beacon & Contest

Automatic beacon with programmable interval. Contest mode with auto-incrementing serial number saved to NVS.

Getting Started

Installation

1

Clone the repository

git clone https://github.com/mfalco1959/IW1RMM_CW_Encoder.git
2

Configure TFT_eSPI

Copy docs/User_Setup_example.h to your TFT_eSPI library folder as User_Setup.h.

#define ILI9341_DRIVER
#define TFT_MISO  12
#define TFT_MOSI  13
#define TFT_SCLK  14
#define TFT_CS    15
#define TFT_DC     2
#define SPI_FREQUENCY  40000000
3

Copy lv_conf.h

Copy include/lv_conf.h into your Arduino IDE LVGL library folder.

4

Arduino IDE settings

Board: ESP32 Dev Module  ·  Flash Mode: DIO  ·  Flash Frequency: 40 MHz  ·  Upload Speed: 115200

⚠ DIO + 40 MHz are required for stability with ZBIT flash chips common on CYD boards.

5

Flash with esptool (alternative)

python -m esptool --chip esp32 --port COM4 --baud 115200 \
  --before default-reset --after hard-reset \
  write-flash -z --flash-mode dio --flash-freq 40m \
  --flash-size detect 0x10000 IW1RMM_CW_Encoder_v714.bin
